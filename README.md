### 소개
☃️ 게임 소개

"눈사람이 녹으면, 봄이 오겠지. 그때 다시 놀자."

겨울이 깊어질수록, 엄마의 숨도 가빠졌다.
창밖에는 여전히 눈사람이 서 있었고, 끝없는 겨울이 계속될 것만 같았다.

엄마는 창문 너머 하얀 세상을 바라보며 조용히 숨을 골랐다.
작은 아이의 눈에, 눈사람은 그녀와의 모든 추억이었다.

하지만 눈사람이 녹아야, 봄이 올 것이다.
